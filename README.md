# eecs-4461

## How to Run:

1. Create python virtual environment:
   - `python -m venv venv`
2. Run python virtual environment:
   - `.\venv\Scripts\Activate`
3. Install Dependencies:
   - `pip install -r requirements.txt`

- To run:
  - cd into src directory (`cd src/`)
  - run: `solara run app.py`
